## Клонирование общего репозитория на компьютер

Чтобы загрузить весь репозиторий на свой компьютер, выполните следующие шаги:

- Откройте терминал (или командную строку на Windows).
- Перейдите в ту папку, в которой хотите сохранить репозиторий. Например, если хотите сохранить его на рабочем столе, используйте команду (вместо Desktop - путь до нужной папки):

```bash
cd ~/Desktop
````

* Создайте SSH-ключ:

```bash
ssh-keygen
```

* Протыквайте клавишу ENTER.
* В выводе команды будет ссылка, где хранится публичный ключ (public key).
* Выведите публичный ключ в консоль командой:

```bash
cat < path_to_your_public_key >
```

* Далее скопируйте этот ключ и откройте GitHub.
* Нажмите на **New SSH key**.
* Вставьте ваш скопированный публичный SSH-ключ в раздел **Key** — он должен начинаться с `ssh` и быть в одной строке.

**Пример:**

```
ssh-ed25519 AAAAC3NzaC1lZDI1NTE5AAAAIP5n3Lkmy1XXt9APDmgYmHx0DKbZNWJZmNPZzx+IBvI6
```

* Вставьте следующую команду для клонирования репозитория:

```bash
git clone https://github.com/CSS-Research-Seminar-25-26/css_research_seminar_25-26
```

* После выполнения команды на вашем компьютере появится папка с репозиторием, содержащая все материалы курса.

### Подгрузить изменения в исходном репозитории

1. Откройте терминал (командную строку)

2. Перейдите в папку с репозиторием:

```bash
cd ~/Desktop/research_seminar_css_2024_2025
```

Выполните команду:

```bash
git pull
```
